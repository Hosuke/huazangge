# 华藏阁

**儒释道三教经典知识库**

基于 [LLMBase](https://github.com/Hosuke/llmbase) 构建的中华传统经典 AI 知识库。
自动从 [CBETA](https://cbeta.org) 和 [ctext.org](https://ctext.org) 获取原典，由 LLM 编译成结构化 wiki，支持智能问答与知识探索。

</div>

## 在 Claude Code 中使用（MCP）

一行命令安装，让 Claude 直接查询华藏阁的 520+ 篇经典文章：

```bash
# 1. 克隆知识库
git clone https://github.com/Hosuke/huazangge.git

# 2. 安装依赖
cd huazangge && pip install -e ../llmbase 2>/dev/null || pip install llmbase

# 3. 添加 MCP server
claude mcp add huazangge -- python -m tools.mcp_server --base-dir $(pwd)
```

重启 Claude Code 后即可使用：

```
> 搜索华藏阁里关于"般若"的文章
> 华藏阁里孟子四端说讲了什么？
> 比较道德经和庄子的核心思想
```

完全本地运行，不消耗服务器资源。

## 特性

- **渐进式学习** — 每次运行自动获取下一批经文，日积月累吸收整部大藏经
- **三教合一** — 支持儒家（四书五经）、佛教（CBETA 大藏经 4,868 部）、道教经典
- **LLM 编译** — 自动提取概念、生成 wiki 文章、建立交叉引用
- **智能问答** — 基于知识库的 Q&A，答案自动归档回 wiki
- **知识图谱** — 可视化经典间的概念关联

## 快速开始

```bash
# 1. 先安装 llmbase
git clone https://github.com/Hosuke/llmbase.git
cd llmbase && pip install -e . && cd ..

# 2. 克隆华藏阁
git clone https://github.com/Hosuke/huazangge.git
cd huazangge

# 3. 配置 LLM
cp ../llmbase/.env.example .env
# 编辑 .env 填入你的 API key

# 4. 开始学习
llmbase ingest cbeta-learn --batch 5       # 学习 5 部佛经
llmbase ingest ctext-book 论语 /analects/zh  # 导入论语
llmbase compile new                         # 编译成 wiki
llmbase web                                 # 启动 Web UI
```

## CBETA 渐进式学习

```bash
# 每次跑一批，自动接续上次进度
llmbase ingest cbeta-learn --batch 10

# 指定部类
llmbase ingest cbeta-learn --category agama     # 阿含部
llmbase ingest cbeta-learn --category bore       # 般若部
llmbase ingest cbeta-learn --category huayan     # 华严部
llmbase ingest cbeta-learn --category chanzong   # 禅宗部
llmbase ingest cbeta-learn --category jingtu     # 净土宗部

# 指定经文
llmbase ingest cbeta-work T0235    # 心经
llmbase ingest cbeta-work T0262    # 法华经
llmbase ingest cbeta-work T0945    # 楞严经

# 查看进度
llmbase ingest cbeta-status

# 12 小时批量学习
python ingest_cbeta_batch.py --hours 12 --batch 20
```

